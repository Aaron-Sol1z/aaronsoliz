SKILLS

HTML5, CSS, and JavaScript are languages that I use as a front-end developer to create maintainable and organized applications. I have experience in using flexbox to help organize pages, making use of the Document Object Model, and making applications that use web APIs to gather information. I also have experience writing code in Java and C for command-line interface projects, as well as familiarity with SQL and NoSQL functions. Primarily, I use Visual Studio Code to work on projects alongside its built-in terminal, debugging tools, and plug-ins such as live servers and access to applications such as GitHub and MongoDB.

Before I type anything, I break down projects into smaller tasks and build up the application little-by-little. This helps me ensure that I understand the requirements of the project and am meeting all goals. Tools such as Git help me keep track of my progress. I also have used Jira in the past for the same purpose of keeping organized. Then I generate test cases that help me ensure the quality of my code and expose elements that need to either be fixed or changed.

When working on a project, I make it a priority to ensure that a quality solution that meets requirements is found.
